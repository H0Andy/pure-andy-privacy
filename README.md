**App Name:** pure-andy-privacy  
**Developer:** Independent Developer (Andy)  
**Contact Email:** leiz11904@gmail.com

This privacy policy explains how our application ("the App") handles user data. We take privacy seriously, especially for young users (ages 4+).

### 1. Current App Status: Fully Offline & No Tracking

The current version of the App operates **completely offline** with no internet-based data transmission.

- **Zero Data Collection:** We do **not** collect, store, process, or share any personally identifiable information (PII), including but not limited to: name, email address, phone number, device ID, IP address, location, or advertising identifier.
- **No Third-Party Services:** The App contains **no** third-party SDKs, libraries, or frameworks for advertising, analytics, crash reporting, or any other tracking purpose.
- **Strictly Local Storage:** All data generated by the App (e.g., game progress, high scores, settings preferences) is saved **only** on your device's internal storage. We cannot access this data remotely.
- **No Internet Permission:** The App does not request or require internet access permission to function.

### 2. Future Feature Disclosure (Optional & User-Controlled)

In future updates, we may introduce **optional** online features (such as a global leaderboard). If implemented:

- These features will be **opt-in only** (you must explicitly agree to participate).
- We will **only** collect:
    - A non-identifiable nickname (user-provided, can be any pseudonym)
    - Basic game score data (e.g., completion time, puzzle difficulty)
- **No** real names, email addresses, or location data will be collected.
- Participation will be completely voluntary, and you can opt out at any time by disabling the feature in settings.
- This policy will be updated and republished before any such feature is released.

### 3. Children's Privacy (COPPA & GDPR-K Compliant)

The App is designed for users aged **4 and above** and fully complies with children's privacy regulations, including COPPA (USA) and GDPR-K (Europe).

- **No Social Features:** The App contains no social networks, chat rooms, messaging, comment sections, or user-to-user communication.
- **No Personal Data from Children:** We do not knowingly collect any personal information from children of any age.
- **No Deceptive Elements:** The App has no hidden in-app purchases, misleading links, external advertisements, or data collection mechanisms disguised as game features.
- **Parental Assurance:** Parents can rest assured that the App is a safe, closed environment with no external data exposure.

### 4. Data Security

Although we collect no data, the local storage on your device is protected by your device's operating system security. We recommend keeping your device updated and using standard security practices.

### 5. Changes to This Policy

If we make changes to this privacy policy, we will update the "Last Updated" date at the top of this document. For material changes (e.g., adding data collection), we will provide a prominent in-app notice.

### 6. Contact Us

If you have any questions, concerns, or requests regarding privacy, please contact the developer:

- **Email:** leiz11904@gmail.com
- **Repository/Project Reference:** pure-andy-privacy
